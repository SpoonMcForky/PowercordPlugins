# IP-Data
Plugin to grab data from the [ipapi](https://ipapi.co/) API for a given IP Address.  

Contains an ``ip`` command  
``.ip {ip}`` Returns data on a given IP Address
## Current Data Sent
- IP
- City
- Region
- Country
- Country Name
- Country Capital
- Country Population
- Postal Code
- Latitude
- Longitude
- Timezone
- Currency Name
